# Виджет банковских операций

## Описание
Приложение для обработки данных о банковских картах и счетах.
Проект включает модули для маскировки номеров, форматирования дат и фильтрации транзакций.

## Функционал
- Маскировка номеров карт и счетов.
- Форматирование дат для удобного отображения.
- Фильтрация операций по статусу (EXECUTED, CANCELED и т.д.).
- Сортировка списка операций по дате (от новых к старым).

## Установка
1. Клонируйте репозиторий: 
   `git clone git@github.com:RomanKeter/Banking_widget.git`
2. Установите зависимости: 
   `poetry install`
